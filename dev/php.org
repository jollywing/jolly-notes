* 面向对象
** 从关联数组生成对象
#+BEGIN_SRC php
<?php
$obj = (object) array('foo' => 'bar', 'property' => 'value');
echo $obj->foo; // prints 'bar'
?>
#+END_SRC

1. 强制类型转换, =(type)expression=.
2. 数组定义, ~array('key' => 'value', 'key' => 'value', ...)~;
3. 对象访问成员, ->.
